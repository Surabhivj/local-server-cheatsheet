# local-server-cheatsheet

**Get connected to server**

```
ssh username@server_address
```
**Copy files from local machine to server**

```
scp -r folder_name username@server_address:/target_directory
```
**Remove directory**
If empty:
```
rm -d dirname 
rmdir dirname
```

non empty
```
rm -r dirname
```
**Remove file**
```
rm filename 
```
